

# KeyClockify 관리하기 

keycloackify 레포를 fork해서 만든 레포지토리입니다.

yarn을 활용하여 install을 진행해주세요.

# storybook 에 추가하기 

storybook에 추가하고 싶은 ft이 생기면 

```
npx keycloakify add-story # Select login.ftl (for example)
```

해당 명령어를 통해 원하는 story를 select 할 수 있습니다 😊
